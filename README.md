# Funnel Analytics & A/B Testing

Analyze e-commerce user journeys, compute funnel metrics, and run A/B test simulations using Matplotlib.

## Features
- Funnel conversion metrics (homepage → product → cart → checkout)
- A/B test simulation with statistical significance
- Interactive Matplotlib visualizations

## How to Run
1. Clone:
   git clone https://github.com/Hardiktyagi9/Funnel-Analytics.git
2. Install dependencies:
   pip install -r requirements.txt
3. Run notebooks in order:
   python notebooks/01_load_&_clean.py
   python notebooks/02_funnel_analysis.py
   python notebooks/03_AB_test_simulation.py

## Tech Stack
Python | Pandas | NumPy | Matplotlib | SciPy
